# Web- ja hybriditeknologiat mobiiliohjelmoinnissa (syksy 2025)

Tässä repositoriossa on kurssin aikana olleet palautettavat viikkotehtävät, sekä luennoilla tehdyt harjoitustehtävät.

## Viikkotehtävät
1. Tavoitesyke  
2. Modal
3. Mdnavigation
4. Todo
5. Reducertodos
6. Barcodereader
7. WeatherApp (datan näyttäminen mobiilisovelluksessa)
8. JoulupukinKontti (Firebasessa db ja auth, datan haku, lisäys ja poisto)
